На форме «Режим администратора», можно нажать на кнопку «Создать словарь понятий»,
следом откроется форма, которая дает администратору функции для создания его собственного словаря.
Так же на форме «Режим администратора» есть кнопка «Редактировать словарь понятий» нажав на которую,
сначала откроется форма для загрузки из файла с расширением .txt, после выбора файла откроется форма и в ней будут загружен словарь для редактирования.
